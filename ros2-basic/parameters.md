# Parameters

* Global Parameters
* Key=Value
* Dynamic reconfigure

```bash
ros2 param list

ros2 param describe <node> <param>

ros2 param get <node> <param>

ros2 param set <node> <param> <value>

ros2 param dump <node> >> <node.yaml>  # save configer in a file
ros2 param load <node> <node.yaml>   # load the configer file
```
